# Projeto - Laboratório de Terminal

## História de Usuário

Como estudante de ADS,
quero utilizar comandos básicos de terminal,
para conseguir organizar arquivos, navegar entre diretórios e manipular dados de forma eficiente.

## Funcionalidades

- Navegação entre diretórios com comandos
- Criação e manipulação de arquivos
- Uso de grep para busca
- Uso de permissões com chmod
- Geração de hash com sha256sum

## Tecnologias

- Bash (terminal)
- Git Bash / Linux Shell
